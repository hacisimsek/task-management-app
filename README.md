# Task Management Application

A React-based task management application with a Java backend.


## Run The Spring Boot Project

```bash
  mvn spring-boot:run
```
## Run Docker Compose File

```bash
  docker-compose up -d
```
## Run Task Management UI

```bash
  npm run dev
```
